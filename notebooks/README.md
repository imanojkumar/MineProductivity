# Notebooks

## Purpose

Jupyter notebooks implementing the pedagogical progression defined by the Learning & Benchmark Suite v1.0, from beginner onboarding through research-grade exploration.

## Scope

Notebook content and its supporting README documentation. Does not include the underlying `mineproductivity` package implementation.

## Responsibilities

- Provide a graduated learning path for new users and contributors.
- Demonstrate MineProductivity capabilities once implemented, at increasing levels of sophistication.

## Contents

- `beginner/` — first-contact notebooks assuming no prior MineProductivity knowledge.
- `intermediate/` — notebooks assuming familiarity with core concepts.
- `advanced/` — notebooks covering multi-package workflows.
- `expert/` — notebooks covering extension, plugin authoring, and performance tuning.
- `research/` — exploratory notebooks for research use cases, not guaranteed to be stable.

## Dependencies

`jupyter` (to be added once notebooks contain runnable code); currently none.

## Future Work

Author notebooks per tier as corresponding subsystems are implemented.
