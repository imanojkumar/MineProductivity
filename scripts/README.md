# Scripts

## Purpose

Operational tooling for bootstrapping, developing, releasing, and quality-checking MineProductivity.

## Scope

Standalone operational scripts. Does not include application/package code (see `src/mineproductivity/`).

## Responsibilities

- Provide repeatable, documented entry points for common maintainer/contributor operations.

## Contents

- `bootstrap/` - first-time environment setup scripts.
- `development/` - day-to-day development helper scripts.
- `release/` - release-cutting and versioning scripts.
- `quality/` - linting, type-checking, and quality-gate scripts.

## Dependencies

Shell/Python, per script, once implemented.

## Future Work

Add scripts as operational needs arise during implementation.
